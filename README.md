# AMS IT System

Система управления актами приёма-передачи техники, с учётом активов, ремонтов и уведомлений.

## Возможности

- Управление техникой (добавление, редактирование, удаление)
- Учет активов и их состояния
- Отслеживание ремонтов
- Система уведомлений
- Различные роли пользователей
- Экспорт данных в CSV
- Интеграция с GLPI

## Технологии

- **Backend**: Node.js, Express, MySQL
- **Frontend**: React, TypeScript, Material-UI
- **Аутентификация**: JWT
- **Безопасность**: bcryptjs

## Быстрый старт

1. Клонируйте репозиторий
2. Следуйте инструкциям по установке в [документации](docs/installation.md)
3. Запустите приложение:
```bash
npm run dev:full
```
